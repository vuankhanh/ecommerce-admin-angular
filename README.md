# Ecommerce Admin Angular

Quản trị hệ thống bán hàng với Angular.

## Project Overview

- Quản lý sản phẩm, đơn hàng, người dùng, báo cáo.
- Giao diện hiện đại, responsive, dễ sử dụng.
- Tích hợp SonarQube kiểm tra chất lượng mã nguồn.
